# Kiosk hero photography

Drop a single landscape JPEG named `coastline.jpg` here. Recommended:

- 2880×1620 (or larger), JPEG, ≤ 600 KB after compression
- Pareus / Caorle coastline, golden hour, no people
- Self-hosted (do not link out — the kiosk may sit on a captive network)

Until a real photo is provided, the welcome screen falls back to its
gradient overlay alone — visually fine but less hospitality-grade.
