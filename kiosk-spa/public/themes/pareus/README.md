# Pareus Beach Resort — kiosk theme assets

Drop the property's brand assets here. The kiosk loads them at runtime from
`/themes/pareus/...`.

## Required

- **`hero.jpg`** — Adriatic coastline / resort photography (golden hour, no
  people). 2880×1620+, JPEG, ≤ 600 KB.
- **`logo.svg`** — Pareus wordmark. Header height h-8.

Until both are provided, the welcome screen falls back to the brand gradient
alone.
